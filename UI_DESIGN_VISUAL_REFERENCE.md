## Color Scheme

### Primary Colors
- **Hero Blue**: `#2f85ff` - Primary action buttons, accents
- **Navy Dark**: `#0a2a6a` - Form focus, secondary actions
- **Background Gradient**: `#0f2f80` ? `#020818` (radial)

### Secondary Colors
- **Light Gray**: `#f3f4f6` - Input backgrounds
- **Dark Text**: `#111827` - Primary text on light backgrounds
- **Muted Text**: `#6b7280` - Helper text
- **White**: `#ffffff` - Card backgrounds, text on dark

### Semantic Colors
- **Success**: `#10b981` (implicit)
- **Error**: `#dc2626` - Validation errors
- **Warning**: `#f59e0b` (implicit)
- **Info**: `#3b82f6` (implicit)

## Responsive Breakpoints

### Mobile (< 768px)
- Hero title: `2.5rem` clamp to `4rem`
- Single column layouts
- Full-width cards with padding
- Stacked buttons

### Tablet (768px - 1024px)
- Hero title: `3rem` clamp to `4rem`
- 2-column grids where applicable
- Optimized spacing

### Desktop (> 1024px)
- Hero title: `4rem` full size
- Multi-column layouts
- Wider cards with constraints
- Side-by-side buttons

---

## Animation Effects

### Transitions
- All elements: `transition: all 0.3s ease`
- Buttons: Hover lift, shadow enhancement
- Cards: Subtle lift on hover
- Inputs: Focus state transition

### Timing
- Hover effects: 0.3s
- Focus states: Instant
- Page transitions: 0.2-0.3s

### Effects
- Lift: `translateY(-2px)`
- Shadow growth: Enhanced on hover
- Color changes: Smooth gradient transitions
- Backdrop: Blur changes on focus

## Accessibility Features

### Focus States
- Keyboard navigation: Clear focus outlines
- Focus color: Yellow (`rgba(255,196,0,0.55)`)
- Outline offset: `2px`

### Color Contrast
- Text on dark: WCAG AA compliant
- Text on light: WCAG AAA compliant
- Icons: Sufficient contrast maintained

### Semantic HTML
- Proper heading hierarchy
- Form labels associated with inputs
- ARIA labels where needed
- Button types properly defined

### Mobile Considerations
- Touch-friendly button sizes (44px minimum)
- Readable font sizes on mobile
- Responsive input groups
- Clear visual feedback

## Performance Considerations

### CSS Optimization
- No animation lag on 60fps devices
- Hardware-accelerated transforms
- Minimal repaints and reflows

### Image Optimization
- Logo: SVG or high-DPI PNG
- Icons: Font-based (Bootstrap Icons)
- Gradients: CSS-based (no images)

### Loading
- Critical CSS: Inline where possible
- Non-critical: Async loading
- Fonts: Web-safe or system fonts
